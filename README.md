# Leviatist.github.io
## Welcome!
该repo是Lev的个人主页。
### 各文件夹介绍
+ script文件夹：
    + 放了一些脚本文件，目前有：搜索脚本，精灵图脚本，位置同步脚本
+ static文件夹：
    + 放了一些静态资源，目前主要是：icon和精灵图
+ style文件夹:
    + 放了一些格式的css定义
    + 页面里面看到的东西都能在格式定义里面找到
    + 命名都是基于语义命名的，应该好找
    + zones里面放子页面
+ index.html：
    + 网页的主页面
